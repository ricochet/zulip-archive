Topic: What is it?

Alex Vidal (Nov 28 2020 at 18:45):

fastlike.dev is a Go emulator/implementation of the Fastly Compute runtime. It allows you to compile wasm binaries that are intended to run on Fastly and run them locally, using fastlike as a proxy or a test suite.
