Karel Hrkal (kajacx) (Jul 17 2023 at 10:37):

Ok, I have found (and even made) some example projects, so here is the list that I currently have for using the component model:

  • wit bindgen examples: My example project on using the component model without wasi both in rust and on the web.
  • MediosZ's component-model-demo: Similar project, includes python runtime, doesn't include web runtime, and doesn't have wit imports. Also the rust runtime is using the wasi_snapshot_preview1.wasm adapter, even though the plugin is build for wasm32-unknown-unkown target, so I don't really understand what is going on there.
  • DougAnderson444's wit-wasm - This contains a full example of wasi components both in Rust runtime and on the web. I have yet to clone the repo and replicate the use cases though.
  • DougAnderson444's rollup-plugin-wit-component - This is an interesting project about "running jco on the web", which allows transforming the wasm component at runtime. It includes wasi support.
