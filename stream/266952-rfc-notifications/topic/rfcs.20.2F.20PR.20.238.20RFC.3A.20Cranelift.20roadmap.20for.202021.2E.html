Rendered RFC

This RFC describes a roadmap for work on Cranelift in 2021 ("and beyond", as we are not likely to complete it all; but better to set goals aspirationally, IMHO, and there is much work to do!).

The major priorities are (i) finish the backend migration, (ii) work to systematically verify and improve correctness, (iii) improve performance, and (iv) investigate some ways to improve and support high-security applications.

The work ranges from concrete, straightforward continuations of current efforts (x86 backend migration) to more ambitious new projects (generate lowering code more systematically, investigate use of VCode as machine-independent IR). The RFC categorizes work by concreteness and by how certain we are that we will be able to complete it this coming year.

Thanks to various folks for feedback on drafts of this; I look forward to lots of continued discussion on all of the work proposed here!
