Topic: Unofficial builds of Python

Brett Cannon (Mar 28 2023 at 00:10):

As part of my (slow) march towards making WASI a tier-2 platform for CPython (which is the Python implementation you get at python.org), I created https://github.com/brettcannon/cpython-wasi-build . It takes less than 5 minutes to produce a build, so I plan to do them as soon as the git tag is produced for a release of CPython (and I remember I need to run the workflow :sweat_smile:).
