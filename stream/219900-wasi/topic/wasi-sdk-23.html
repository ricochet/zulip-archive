Alex Crichton (Jul 17 2024 at 22:42):

I've deleted the old wasi-sdk-23 tag (was never fully published on github releases) and re-tagged the latest commit on main as wasi-sdk-23. This tag has a few notable bug fixes and additionally contains a refactoring of the build system of wasi-sdk itself.

If folks have a moment you can test out the release binaries in their pre-release state and report any issues. The thinking is to publish this officially after a few days of testing.

One notable change is that the names of artifacts are now different, so scripts will need to be updated with more than just a version number change if there are existing scripts to download wasi-sdk.
