Extra Info

Cranelift uses R_AARCH64_TLSGD_ADR_PAGE21 and R_AARCH64_TLSGD_ADD_LO12_NC relocations for TLS, which seems to be unsupported by lld. Instead I think TLSDESC has to be used.
